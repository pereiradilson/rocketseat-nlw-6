## Prerequisites

- [Node LTS](https://nodejs.org/en/)
- [Yarn](https://classic.yarnpkg.com/pt-BR/)

## Cloning project

```bash

  # Clone repository
  $ git clone https://github.com/pereiradilson/rocketseat-nlw-6.git

  # Enter folder
  $ cd rocketseat-nlw-6

```

## Install and run

```bash

  # Install
  $ yarn

  # Run
  $ yarn start

```
